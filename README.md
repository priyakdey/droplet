# Droplet

![Droplet](./ui/src/assets/droplet.svg)

**Droplet** is a minimal, secure, and modern file-sharing application built as a full-stack educational project.

## 🌐 Tech Stack
- **Frontend**: React + TypeScript + TailwindCSS (shadcn/ui)
- **Backend**: Spring Boot 3 (Java 23)
- **Storage**: Azure Blob Storage (Dockerized local setup for development)
- **Auth**: Username/Password (Spring Security)

## 🚧 Status

Still under heavy development — core UI and API contracts are being set up.  
Production deployment and Azure integration to follow.

## 🧠 Purpose

This is a purely educational full-stack playground — built to learn and apply modern design, architecture, and devops best practices.

Stay tuned! 🚀
